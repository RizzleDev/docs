# Leveling Commands

| Command      | Description                             | Usage |
| :----------- | :-------------------------------------- | :---- |
| /leaderboard | Shows the top leveling users            |       |
| /level       | Shows your level information            |       |
| /rank        | Shows your or a user rank in a rankcard |       |
